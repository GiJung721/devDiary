# 마크다운 문법(Syntax)

## 1. 제목(Header)

제목은 `<h1>` 부터 `<h6>` 까지 사용할 수 있다. 숫자가 커질수록 글자 크기는 작아진다.

```markdown
# 제목 h1
## 제목 h2
### 제목 h3
#### 제목 h4
##### 제목 h5
###### 제목 h6
```

* 예시

# 제목 h1

## 제목 h2

### 제목 h3

#### 제목 h4

##### 제목 h5

###### 제목 h6

____



# 2. 인용문 사용해서 텍스트 박스 강조하기

```markdown
> **NOTE: ** this is note content.
> note example.
```

* 예시

  > **NOTE: ** This is note content.
  >
  > note example.
